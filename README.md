## Descripción General

El esquema `tourism` ha sido diseñado para la gestión de alojamientos turísticos y reservas. Permite administrar información relacionada con alojamientos, habitaciones, huéspedes, reservas, pagos, ubicaciones, servicios y valoraciones realizadas por los usuarios.

## Relaciones Principales

- Un propietario puede administrar múltiples alojamientos.
- Un alojamiento puede poseer varias habitaciones.
- Los huéspedes pueden realizar múltiples reservas.
- Las reservas pueden incluir varios huéspedes.
- Los pagos se registran y vinculan a las reservas correspondientes.
- Los alojamientos pueden contar con diversas amenidades.
- Los huéspedes pueden registrar reseñas y valoraciones.
- Las ubicaciones permiten organizar geográficamente los alojamientos.
